“Ethernet modules” screen
   
   

Screen presents the list of Ethernet modules though which communication with the alarm control panels registered in the INTEGRUM system takes place.

At the top of the screen, the Menu bar is displayed.

[region name] - information about the region selected in the regions menu. The screen displays modules connected to the control panels belonging to the given region (if the “Show child items” option is enabled, modules connected to the control panels belonging the sub-regions are shown too). Click on the region name to display the regions menu and select another region.

Name - name of the Ethernet module in the INTEGRUM system.

Address - address of the Ethernet module (IP address or domain name).

GUARDX port - number of the TCP port used by INTEGRUM for communication with the alarm control panel.

Control panel - name of the alarm control panel to which the Ethernet module is connected. Click on the control panel name to display the screen with its data (see Control panel data screen).

Status indicators - information about the module status presented by status indicators. Each column is designated by an icon for identification of the status indicators and filtering of the modules by status (see Filtering by Ethernet module status). Action - the following button is displayed:

- click to display the screen with Ethernet module data (see Ethernet module data screen).

- click to filter the list by predefined criteria (see Filtering the list of Ethernet modules).

- click to remove the filter and display the list of all modules.

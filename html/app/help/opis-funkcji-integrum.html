Description of functions available to INTEGRUM application users
   
   

Access to dashboard - the function allows you to access the “Dashboard” screen.

Operating map items - the function allows you to operate the items in the “Map” window on the “Dashboard” screen.

Commenting on new events - if the user has access to the function, the Commenting window is displayed on the “Dashboard” screen to allow you to make comments on new events of alarm or trouble type.

Downloading map editor - if granted access to the function, the user can download the MAP EDITOR program by using the button in the “Map” window on the “Dashboard” screen.

 

Viewing events - the function provides access to the “Event log” screen.

Adding comments to events - the function allows you to add comments to the events displayed on the “Event log” screen and in the “Events” windows on the “Dashboard” screen.

 

Viewing INTEGRA control panels - the function provides access to the “Control panels” screen.

Adding INTEGRA control panels - the function allows adding control panels to the INTEGRUM system.

Editing INTEGRA control panels - the function allows you to edit settings of the INTEGRA control panels registered in the INTEGRUM system.

Deleting INTEGRA control panels - the function allows removing control panels from the INTEGRUM system.

Operating INTEGRA control panels - the function allows you to operate control panels registered in the INTEGRUM system.

Exporting INTEGRA control panels - the function allows exporting the list of control panels to CSV file.

Configuration supervision - the function allows access to the “Configuration supervision” tab on the screen with alarm control panel data.

Viewing INTEGRA users - the function allows access to the “Users” tab on the screen with alarm control panel data.

Adding INTEGRA users - the function allows adding users to the alarm control panels.

Editing INTEGRA users - the function allows you to edit users of the alarm control panels.

Granting rights without using schedules - the function allows editing the type and rights of the control panel users (without using the user schedules).

Deleting INTEGRA users - the function allows removing users from the alarm control panels.

 

Viewing INTEGRUM users - the function allows access to the “Users” screen.

Adding INTEGRUM users - the function allows you to add INTEGRUM users.

Editing INTEGRUM users - the function allows you to edit INTEGRUM users.

Deleting INTEGRUM users - the function allows you to delete INTEGRUM users.

Managing application access - the function allows you to manage access to the INTEGRUM application.

 

Managing INTEGRA user schedules - the function allows you to access the “User schedules” screen, manage the user schedules and use the schedule when adding and editing the alarm control panel users.

 

Managing licenses - the function allows managing licenses on the “Status” screen.

 

Managing regions - the function allows access to the “Region settings” screen.

 

Managing roles - the function allows you to access the “User roles” screen and manage the user roles.

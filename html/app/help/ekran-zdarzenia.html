“Event log” screen
   
   

The screen presents the list of events read from the control panels registered in the INTEGRUM system or generated by the INTEGRUM system (e.g. establishing connection or no communication with the control panel).

The Menu bar is displayed at the top of the screen.

- click to export events to CSV file.

- click if you want to use colors to differentiate between the event types. The button is displayed when events are not highlighted in colors.

- click if you do not want to use colors to differentiate between the event types. The button is displayed when events are highlighted in colors.

[region name] - information about the region, events from which are displayed. Events from the INTEGRUM system are only displayed for the global region, covering all regions. Click on a region name to display the regions menu (you can click on another region to display events from that region).

Date - date and time when event occurred.

Source - name of the alarm control panel the event relates to. Click on the name of the control panel to display the screen with its data (see Control panel data screen).

Type - icon indicating the type of event:

 - alarm from zone (including tamper),

 - another alarm,

 - arming / disarming, alarm clearing,

 - zone bypassing,

 - access control,

 - diagnostics,

 - function used,

 - system event,

 - configuration supervision,

 - communication with the system.

Event - event description. Next to the description, the button is displayed. Click on to open the Comments window. If the event has already been commented, a number corresponding to the number of comments is displayed instead of the + symbol.

User - name of INTEGRUM user or INTEGRA alarm control panel user (if the user is not linked with any INTEGRUM user).

- click to filter the list by predefined criteria.

- click to remove the filter and display the list of all events.

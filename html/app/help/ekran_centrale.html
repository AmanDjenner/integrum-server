“Control panels” screen
   
   

The screen presents the list of INTEGRA / INTEGRA Plus control panels registered in the INTEGRUM system and allows you to manage them. You can also use this screen to arm/disarm the alarm systems.

The Menu bar is displayed at the top of the screen.

 - click to add a new control panel to the INTEGRUM system (see Adding a new control panel).

- click to export the list of control panels to CSV file.

[region name] - information about the region selected in the regions menu. Control panels belonging to the given region are displayed on the screen (if the “Show child items” option is enabled, control panels belonging to sub-regions are shown as well). Click on the region name to display the regions menu and select another region.

Name - name of the alarm control panel in the INTEGRUM system. Click on the control panel name to display the control panel data screen (see Control panel data screen).

Address - address of the Ethernet module through which communication with the control panel takes place.

Status indicators - information about the alarm system status is presented by means of status indicators. The status indicators perform a similar function as the LEDs in keypads. Each column is designated by an icon for identification of the status indicators and filtering of the control panels by status (see Filtering by control panel status). How the status is presented in each column is explained in the table below. Click on the status indicator to go to the “Event log” screen. Events related to the given control panel and the status presented in the column will be displayed:

- alarms,

- troubles,

- arming / disarming events,

- system events,

- events related to communication between INTEGRUM and control panel.

Which data are presented, which partitions are supported etc. depends on the Ethernet module settings (virtual keypad parameters). These settings are configured by installer of the alarm system.

To get further information on the indicators in column , go to the screen with control panel data.

Action - the following buttons can be displayed:

- click to arm all partitions which are disarmed. The button is displayed if there are some disarmed partitions.

- click to disarm all partitions that are armed. The button is displayed if there are some armed partitions.

Which partitions will be armed / disarmed depends on:

- Ethernet module settings (virtual keypad parameters),

- rights of the code used (the service code allows you to operate all partitions, irrespective of the Ethernet module settings).

- click to display the screen with control panel data.

- click to remove the control panel from the INTEGRUM system (see Deleting a control panel).

 - click to filter the list by predefined criteria (see Filtering the list of control panels).

 - click to remove the filter and display the list of all control panels.

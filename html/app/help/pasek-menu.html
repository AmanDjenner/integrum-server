Menu bar
   
   

The menu bar displays the main menu.

- click to display the regions menu.

- click to display the “Users” screen.

- click to display the “Control panels” screen..

- click to display the “Event log” screen..

- click to display the “Dashboard” screen.

- click to display the drop-down menu:

Status - click to display the “Status” screen.

User schedules - click to display the “User schedules” screen.

User roles - click to display the “User roles” screen.

Region settings - click to display the “Region settings” screen.

 - click to display the “Status” screen. The button is displayed if you have no access to the “Managing INTEGRA user schedules”, “Managing licenses” and “Managing regions” functions.

- click to display the “Help”.

- click to display your settings in the INTEGRUM system. Name of the logged-in user is displayed next to the button.

- click to log out.

At some icons, descriptions are displayed. The descriptions are not displayed, if they do not fit on the menu bar (display resolution is too low or window size is too small).

Hiding the main menu

The main menu is hidden if the display resolution is too low or the window size is too small for the whole main menu to fit on the menu bar.

- click to display the main menu (the button is displayed when the main menu is hidden).

“Status” screen
   
   

The screen presents information about the INTEGRUM system status and allows you to manage licenses.

The Menu bar is displayed at the top of the screen.

System components - information about the status of system components is presented with icons. How the status is presented is explained in the table below. Moving the cursor over the icon will display the following information:

if the component is working properly - version of the given component firmware,

if the component is not working - reported problems.

The system components include:

INTGERUM Web - application allowing you to operate the INTEGRUM system,

INTEGRUM AppServer - intermediate component for communication between the other INTEGRUM system components,

INTEGRUM Server - program for data exchange between alarm control panels and INTEGRUM,

INTEGRUM DB - database.

Licence holder - information displayed in this field depends on the license status:

- if no licence has been loaded yet - a message is displayed to inform you that the licence is missing,

- if at least one licence has been loaded, but has not been verified yet - a message is displayed to inform you that the licence has not been verified,

- if at least one licence has been loaded and verified - data of the licence holder are displayed.

The same information is displayed on the login page, under the “LOG IN” button.

Devices - information on Ethernet modules connected to the control panels registered in the INTEGRUM system:

With licence - number of Ethernet modules with licence.

Without licence - number of Ethernet modules without licence.

Unsupported - number of Ethernet modules which are not supported (unsupported firmware version).

Details - click to display the “Ethernet modules” screen.

Licence request - click on “Generate and save” to generate a file containing the data of Ethernet modules without licence (file with SLR extension) and save it. Location where the file will be saved depends on the web browser.

Licence - click on “Browse” to indicate location of the file containing licences for Ethernet modules (file with SLR extension).

Upload - click to upload the licences for Ethernet modules from the indicated SLF file.

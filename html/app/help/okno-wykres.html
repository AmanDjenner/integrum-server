“Diagram” window
   
   

The window presents in graphic form the information on status of the alarm systems registered in INTEGRUM.

The title bar displays:

•name of the region to which information presented in the window relates. Click on the region name to display the regions menu and select another region.

•icons:

- the number next to the icons indicates how many events await comment.

- the number next to the icons indicates how many events await your comment. Click on the icon to display the Commenting window.

• button - click if you want the data displayed in the window to be updated.

At the top of the window, color buttons designated by icon are displayed:

- alarm - the number beside the icons indicates how many alarm systems are signaling alarm,

- trouble - the number beside the icons indicates how many alarm systems are signaling trouble,

- service mode - the number beside the icons indicates how many alarm systems are running in service mode,

- no communication - the number beside the icons indicates with how many alarm systems there is no communication.

Click on the button to display in the “Details” window the list of control panels for which the indicated status is valid.

The diagram presents:

•red color - number of alarm systems which were signaling alarm,

•yellow color - number of alarm systems which were signaling trouble,

•green color - number of alarm systems which were running in service mode,

•blue color - number of alarm systems with which there was no communication.

The diagram presents the latest 30 minutes (at most).
